# IPL Kartika Residence — Web Admin

Aplikasi web untuk manajemen IPL Cluster Kawaluyaan Kartika Residence.

## Fitur

### Untuk Warga
- Daftar akun & setup profil
- Bayar IPL (upload bukti transfer)
- Riwayat pembayaran
- Edit profil & ganti password
- Lihat dokumen pengurus

### Untuk Admin/Superuser
- Dashboard statistik
- Kelola warga (CRUD)
- Kelola tagihan (approve/reject/generate)
- Laporan keuangan
- Pengaturan nominal IPL
- Upload dokumen PDF
- Log aktivitas
- Export Excel semua modul

## Deploy

Deployed on Streamlit Cloud: https://xxx.streamlit.app

## Stack

- **Frontend**: HTML + Vanilla JS + Supabase JS SDK
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Deploy**: Streamlit Cloud
- **Excel Export**: SheetJS (XLSX)
